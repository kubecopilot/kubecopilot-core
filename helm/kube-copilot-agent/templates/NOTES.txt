2. Deploy an agent (or use the github-copilot-agent Helm chart):

   kubectl apply -f - <<MANIFEST
   apiVersion: kubecopilot.io/v1
   kind: KubeCopilotAgent
   metadata:
     name: my-agent
     namespace: {{ .Values.namespace }}
   spec:
     githubTokenSecretRef:
       name: github-token
   MANIFEST

3. Send a message:

   kubectl apply -f - <<MANIFEST
   apiVersion: kubecopilot.io/v1
   kind: KubeCopilotSend
   metadata:
     name: my-question
     namespace: {{ .Values.namespace }}
   spec:
     agentRef: my-agent
     message: "What is the health of this cluster?"
   MANIFEST

4. (Optional) Deploy the Web UI using the kube-copilot-ui Helm chart:

   helm upgrade --install kube-copilot-ui ./helm/kube-copilot-ui \
     --namespace {{ .Values.namespace }}
